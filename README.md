# offilne_calendar
    성균관대학교 2018년 2학기 문제해결과 알고리즘 팀프로젝트입니다.
    해당 학기 이 코드를 통해 성균관대학교 러닝페어 전체 2등을 달성했습니다!!🔥
    
    코딩을 배운지 얼마 안되고 짠거라 부족한점이 많습니다.
    1. 너무나 많은 반복 - 파라미터처리로 인해 부족했었습니다.
    2. 코딩 컨벤션 무시
    3. 무거운 프로그램
    4. 데이터베이스 취약
    5. 분할 정복 없음 (모듈이 없이 한 코드에 전체코드르 다 심음)
    
    조장 - 고준서 (본인)
    팀원 - 정우석, 이준화, 장현수, 김종현, 정현호
